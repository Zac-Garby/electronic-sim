# Electronic Simulation
![Screenshot](img/screenshot.png)

## Getting started:
Run the following commands (You'll need to install NodeJS beforehand):

```
git clone https://github.com/Zac-Garby/electronic-sim
cd electronic-sim
npm install
npm start
```
